# 会员资料前台调用规则

本页用于统一说明会员资料在前台的显示与调用约定，便于主题开发、模块接入和后续排版扩展。

## 公开显示名

- 前台公开显示名统一调用 `public_display_name`
- 取值优先级：
  1. `nickname`
  2. `display_name`
  3. 由 `first_name` / `last_name` 组合生成的昵称
  4. 默认文案 `会员用户`
- 前台列表、评论区、文章作者信息、会员中心头部均不应直接暴露 `username`，除非用户主动选择“使用用户名”作为公开昵称

## 作者简介

- 前台作者简介统一调用 `author_bio`
- 优先读取会员资料页填写的 `bio`
- 若 `bio` 为空，则允许回退到作者扩展简介或默认简介文案
- 当前调用位置包括：
  - 文章详情页作者卡片
  - 作者列表卡片
  - 作者个人页简介区

## 头像尺寸

- 评论区、顶部导航、轻量作者信息默认调用小尺寸头像
- 会员中心左侧栏、作者卡片默认调用中尺寸头像
- 需要高质量展示时可调用大尺寸头像
- 统一通过头像接口获取，避免模板直接拼接文件路径

## 会员资料编辑页原则

- 高频字段尽量集中、紧凑展示
- 公开昵称与昵称方式保持同一行展示
- 低频说明文字不在页面堆叠，详细规则写入文档
- 作者简介字段应控制在适合前台卡片展示的长度范围内
