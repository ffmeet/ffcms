# 系统总览

## 技术栈

- Laravel 12
- Filament 5.3 系列管理后台（`filament/filament: ^5.3`）
- Livewire 媒体与设置工作台
- Tailwind/Vite 前端资源构建
- SQLite（当前本地环境）

## 版本提醒

- 本文档默认以 `Filament 5.3` 的页面注册、表单 Schema、导航和面板 API 为准。
- 如果你当前项目不是 Filament 5.3 系列，而是更低或更高的大版本，开发时不要直接照搬本文档里的后台实现细节。

## 模块划分

### 前台内容层

- 首页、栏目、标签、作者、文章详情
- 活动列表与详情
- 商品列表与详情
- 会员计划页

### 会员中心

- 个人资料
- 我的稿件
- 我的评论
- 我的订单
- 我的订阅
- 活动中心 / 我的活动

### 后台管理

- 文章、分类、标签、内容模型
- 活动、商品、会员计划
- 用户、会员组、订单、支付、评论
- 设置中心、品牌中心、路由中心、缓存中心、首页设置

## 关键入口

- 首页路由：`/`
- 会员中心路由：`/member`
- 后台入口：`/admin`

## 配置中心

站点级配置集中在 `SiteSetting`，并通过 `business_settings` 承载：

- 当前主题
- 首页位置位配置
- 品牌信息
- 支付渠道
- 路由与缓存相关设置

## 缓存结构

- 站点配置缓存：`SiteSetting::current()`
- 前台结果缓存：`FrontendCache`
- 后台提供统一的缓存清理入口
