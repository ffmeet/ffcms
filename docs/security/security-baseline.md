# 安全基线

## 生产环境最低要求

1. `APP_DEBUG=false`
2. 真实 SMTP 已配置
3. 公开 webhook 具备签名校验
4. 禁止生产环境模拟支付
5. 登录、注册、找回密码接口启用限流
6. 后台账号与前台普通账号分离管理

## 已完成的基础项

- 会员能力通过权限中间件控制
- 后台主题预览入口已挂管理员权限
- 封面上传与头像上传具备基础图片校验
- 登录、注册、找回密码、重置密码提交路由已补基础限流
- 生产环境模拟支付已禁用

## 仍需完成的正式支付安全项

1. 渠道签名校验
2. webhook 幂等处理
3. 渠道支付成功与系统回写的一致性校验
4. 异常订单人工处理流程

## 审计建议频率

- 每次发布前做一次快速审查
- 每次支付、上传、权限改动后做专项复核
